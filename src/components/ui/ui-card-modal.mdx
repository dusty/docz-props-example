# UiCardModal

Wrap any element in a card modal. The element will be mounted and animated when **isOpen** is true.

The footer will always show the **Cancel** button. The **Save** and **Delete** buttons are only shown if the **onSave** or **onDelete** props are passed in.

When the delete icon on the top-right or the **Cancel** button is clicked, it will fire the **onCancel** event. If the **Save** button is clicked, it will fire the **onSave** button. If the **Delete** button is clicked it will fire the **onDelete** button.

The parent component should handle these events and change the **isOpen** prop to false to close the modal.
