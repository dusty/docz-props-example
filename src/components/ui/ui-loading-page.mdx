# UiLoadingPage

UiLoadingPage shows a full-screen page with a loading indicator. There are no props to control visibility, it is expected to replace the rendering of the page based on criteria.

## Example

<Playground>
  <IFramePlayground minHeight={400} maxHeight={800}>
    <UiLoadingPage />
  </IFramePlayground>
</Playground>

### Example Usage

```
const App = () => {
  return isLoading ? <UiLoadingPage /> : <RealComponent />
}
```
